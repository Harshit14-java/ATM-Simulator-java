 ATM Simulator System

ATM Simulator System is a Java-based application that simulates the basic operations of an ATM machine.
It allows users to perform banking operations such as checking balance, depositing money, withdrawing cash, and viewing transaction details.

 Technologies Used

* Java
* Object Oriented Programming (OOP)
* MySQL(for jdbc connection)

 Features

* User login authentication
* Balance inquiry
* Cash withdrawal
* Deposit money
* Transaction details

 Project Description

This project demonstrates the core concepts of Java programming and object-oriented design.
It replicates the basic workflow of an ATM machine and helps in understanding banking transaction logic.
